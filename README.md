# 大促分析决策画布

## 直接打开网页

[点击进入流程图网页](https://2022823124-afk.github.io/xitongjiagou.github.io/)

如果上面的链接暂时打不开，请到仓库的 **Settings → Pages**：

1. **Source** 选择 `Deploy from a branch`
2. **Branch** 选择 `main`
3. 文件夹选择 `/ (root)`
4. 点击 **Save**
5. 等 1-3 分钟后再打开上面的网页链接

## 说明

- GitHub Pages 版本适合展示、单人编辑、本地保存、导入/导出 JSON。
- 如果需要多人实时同步编辑，需要部署 Node 服务版本，例如 Render / Railway，因为 GitHub Pages 不能运行 `server.mjs`。
